Reactive Frontend with RxJS

Sep 7, 2024

RxJS is a library that brings the reactive programming paradigm to Javascript. I used the RxJS library in a web application developed with Angular.

RxJS is an implementation of Observables in Javascript [1]. Reactive and asynchronous programming are really useful for building modern web applications. RxJS simplifies managing streams of data, combining operations, repeating failed operations, and cancel asynchronous operations.

The essential concepts in RxJS are: Observable, Observer, Subscription, Operators, and Subject. An Observable represents a stream of data, or to be more precise an invokable collection of future values. An Observer can subscribe to one Observable in order to receive new values. Operators are functions that generate a new Observable and are used to transform, filter, and combine data. A Subject is a kind of Observable that allows multiple subscribers to receive the same values.

In the following example I created a service that generates values that are displayed asynchronously in a web page.

The class MachineStatus is a data structure that represents the status of a machine:

export class MachineStatus {
  public configName: string;

  constructor(configName: string) {
    this.configName = configName;
  }
}

The class Configuration represents one specific configuration of a machine:

export class Configuration {
  public name: string;
  public finalConfigName: string;

  constructor(name: string, finalConfigName: string) {
    this.name = name;
    this.finalConfigName = finalConfigName;
  }

  public evolve(): MachineStatus {
    return new MachineStatus(this.finalConfigName);
  }
}

The following code defines the service AlgorithmService. The service produces an Observable of type BehaviorSubject. The values of type MachineStatus are added to the Observable using the time interval service.

import { Injectable } from "@angular/core";
import {
  Observable,
  BehaviorSubject,
  interval
} from "rxjs";

@Injectable()
export class AlgorithmService {
  private machineStatus: BehaviorSubject<MachineStatus>;
  private configurations: Configuration[];
  private configuration: Configuration;
  private actualStatus: MachineStatus;

  constructor() {
    const initial = newMachineStatus("s");
    this.machineStatus = newBehaviorSubject(initial);
    this.configurations = [];
    this.configuration = newConfiguration("","");
    this.actualStatus = newMachineStatus("");
  }

  public evolve(
      configurations: Array<Configuration>
    ): Observable<MachineStatus> {
    this.configurations = configurations;
    this.configuration = configurations[0];
    this.actualStatus = new MachineStatus(this.configuration.name);

    this.evolution();
    return this.machineStatus.asObservable();
  }

  private evolution(): void {
    const source = interval(1000);
    source.subscribe(() => {
      this.actualStatus = this.configuration.evolve();
      this.machineStatus.next(this.actualStatus);
      this.configuration = this.configurations.filter(
        c => c.name = this.actualStatus.configName)[0];
    });
  }
}

The following code represents the component TapeComponent. The component initializes a BehaviorSubject object and executes the algorithm service, that will add the generated values asynchronously to the object.

import { Component } from "@angular/core";
import {
  Observable,
  BehaviorSubject,
  take
} from "rxjs";

@Component({
  selector: "app-tape",
  templateUrl: "./tape.component.html"
})
export class TapeComponent {
  private machineStatuses: MachineStatus[]
  private subject: BehaviorSubject<MachineStatus[]>;
  public machineStatus: Observable<MachineStatus[]>;

  constructor(private algorithm: AlgorithmService) {
    this.machineStatuses = []
    this.subject = new BehaviorSubject(this.machineStatuses);
    this.machineStatus = this.subject.asObservable();
    this.evolveAlgorithm();
  }

  private evolveAlgorithm(): void {
    const firstConfiguration = new Configuration("b","o");
    const secondConfiguration = new Configuration("o","o");
    const configurations = [firstConfiguration, secondConfiguration];
    this.algorithm
      .evolve(configurations)
      .pipe(take(5))
      .subscribe(
        (machineStatus) => {
          this.machineStatuses.push(machineStatus);
          this.subject.next(this.machineStatuses);
        }
      );
  }
}

The following HTML snippet is associated to the TapeComponent and shows how the values inserted in the Observable object are retrieved and displayed in the web page.

<div *ngFor="let status of machineStatus | async">
  <span>{{ status.configName }}</span>
</div>
