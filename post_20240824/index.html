Blog Template

Aug 24, 2024

I released the template of the web application I developed for this web site.

You can find the code in the following GitHub repository:
sitMCella/blog-template

The web application uses a custom blog design and is built using React and Tailwind CSS. In addition, the web application uses Prism.js to display the code snippets. The React web application makes use of the Single Page Application model.

The React application has been adjusted for the single page application to be supported in GitHub Pages [1].

A key design choice for this web application is its minimalistic style. The navigation bar at the top of the page features concise menu options, promoting ease of use and a clean interface. The Blog page displays posts using simple, card-based layouts that include the article's headline, publication date, excerpt, and relevant tags. Additionally, a search bar is provided to filter articles based on tags. Users can begin typing a keyword into the text box and then select a matching tag from the dropdown menu.

The article pages maintain a minimalist aesthetic, incorporating panels and code snippets to enhance readability and structure. Links and references may appear either inline within the article or at the bottom of the page, offering flexibility in how supplementary information is presented.

In future I will add new features (for example the dark mode), and I will apply changes on some components of the pages.
