CAD Desktop Application in JavaFX

Sep 4, 2024

In the past I worked with AWT and Swing in Java, and more recently I wanted to experiment with JavaFX creating a simple CAD desktop application.

JavaFX [1] is a toolkit for developing rich client applications, for desktop, mobile, and embedded systems built on Java. Starting from Java JDK 11, JavaFX became part of the OpenJDK project.

For my CAD application I decided to use Java 21 LTS, and Maven for managing the required JavaFX libraries.

JavaFX is a set of graphics and media packages and the main architecture components are the JavaFX API and the scene graph. The scene graph consists of a hierarchical tree of nodes that represents all of the visual elements on the application's user interface. For each element you can apply a state (for example the position on the screen), and effects that change the appearance of the element.

JavaFX provides UI controls, cascading style sheets (CSS), layout containers, transformations, and visual effects for the graphical elements.

Many different libraries are available in order to customise the look-and-feel of your application. I used the Ikonli Material Design 2 [2] for the icons in the application, and atlantafx [3] as the CSS theme for the visual components.

The layout of the application can be composed either using the JavaFX APIs or JavaFX FXML. The second option is useful to separate the UI layout code from the application logic.

I found the JavaFX APIs extremely easy and intuitive to use, and the JavaFX framework provides out of the box support for 2D and 3D graphics, in my case I was interested on the shapes for 2D modelling.

On high level, the JavaFX application is composed by a Stage and a Scene. The JavaFX Stage class is the top level JavaFX container, and represents a window in a JavaFX desktop application. Inside a JavaFX Stage you can add a JavaFX Scene, which represents the content of the window. Apart the root Stage object, you can create additional Stage objects when the application needs to open new windows.

The following is a simple example of a JavaFX application:

import javafx.scene.control.Label;
import javafx.scene.layout.Pane;
import javafx.scene.Group;
import javafx.scene.Scene;
import javafx.scene.shape.Line
import javafx.scene.shape.Shape
import javafx.stage.Stage;

private Pane pane;

public class App extends Application {
  @Override
  public void start(Stage stage) {
    stage.setTitle("Simple CAD");

    Label label = new Label("Label");

    this.pane = new Pane();
    pane.setMinSize(500, 400);
    pane.setOnMouseClicked(this::handleMouseClick);
    pane.getChildren().add(label);

    Group root = new Group(pane);

    var scene = new Scene(root, 600, 500);
    stage.setScene(scene);
    stage.show();
  }

  private void handleMouseClick(MouseEvent event) {
    var line = new Line();
    line.setStartX(event.getX());
    line.setStartY(event.getY());
    line.setEndX(event.getX()+10);
    line.setEndY(event.getY()+5);
    line.setStrokeWidth(1.0);
    this.pane.getChildren().add(line);
  }
}

The Scene contains one Group object with one Pane. In the previous example the shapes are added to the Pane on mouse click event.
