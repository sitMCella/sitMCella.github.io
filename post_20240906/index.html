2D CAD Desktop Application

Sep 6, 2024

I developed a simple 2D CAD desktop application. Computer Aided Design (CAD) applications are used for creating engineering and architectural designs and drawings (to name a few).

The application focuses on drawing two-dimensional shapes in a canvas. The user must create a new Project for starting a new drawing. In addition, the application provides the Save and Open Project features, to store and load the project data.

The application allows the user to create shapes in the canvas using mouse actions. The drawing procedure is initiated on the first mouse click in the canvas (after selecting the icon that corresponds to the shape), and the graphical object is re-shaped on mouse movement. Currently, just the Line and Curve shapes are available.

Each graphical object on the canvas is represented by its shape and the points that define the shape. The user can optionally hide the points associated with the shapes in the canvas. The drawing procedure provides additional features: select the closest point (when the mouse pointer is close to another shape), and draw a straight line for the Line shape. The canvas does not provide a grid.

After a shape has been added to the canvas, the user can select the shape using the mouse click, and access to its properties. From the properties panel the user can update the coordinates of the shape. As an alternative approach, the user can modify the shape re-initiating the drawing procedure with the mouse interaction.

The application does not allow to create layers or groups of objects, however I introduced the concept of Tags for the shapes. The user can define a list of Tags on Project level, and then assign specific Tags to each shape. From the Tags filter panel, the user can select one Tag, as a result all of the shapes that are not associated with the specified Tag are greyed out in the canvas.

Overall, the application is extremely simple, and does not provide all the features that you can expect from commercial CAD applications. In the previous article I talked about some of the technical aspects of the application. This project has been a good exercise to experiment with JavaFX, and to work on the design of the architecture for a CAD application.
